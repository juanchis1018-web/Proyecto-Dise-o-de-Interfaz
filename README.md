# Sistema de Recomendación Académica y Profesional

## Descripción

Este sistema permitirá a los usuarios recibir recomendaciones sobre:

- Qué estudiar
- Qué carrera universitaria elegir
- Qué habilidades desarrollar
- Qué trabajo elegir

Basado en sus:

- Intereses
- Habilidades
- Preferencias académicas
